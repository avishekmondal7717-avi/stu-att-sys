# Smart Face Recognition Attendance System

This repository contains a clean, restructured setup of the Smart Attendance System, featuring a FastAPI backend and a Vite+React frontend.

## Project Structure

```text
dfs-backend/
├── backend/                  # FastAPI Backend
│   ├── data/                 # ONNX models, pickle database files (faces_data.pkl, names.pkl)
│   ├── tools/                # Legacy CLI / Streamlit utilities
│   │   ├── Attendance/       # Local CSV logs generated by test.py
│   │   ├── add_faces.py      # Enroll new faces from terminal
│   │   ├── test.py           # Real-time local camera surveillance & logger
│   │   └── app.py            # Local Streamlit attendance dashboard
│   ├── database.py           # SQLite DB setup and seed data
│   ├── face_engine.py        # Core face recognition & anti-spoofing engine
│   ├── face_service.py       # Wrapper layer for FastAPI endpoint processing
│   ├── main.py               # FastAPI App entrypoint
│   └── requirements.txt      # Python dependencies
└── frontend/                 # Vite + React Frontend
    ├── public/               # Static assets
    ├── src/                  # React source files
    ├── package.json          # Node dependencies
    └── vite.config.js        # Vite config with API proxy
```

## Getting Started

### 1. Run the Backend

Navigate to the `backend/` directory, set up your Python environment, install dependencies, and start the FastAPI server:

```bash
cd backend

# Create virtual environment
python -m venv .venv
# Activate virtual environment (Windows)
.venv\Scripts\activate

# Install requirements
pip install -r requirements.txt

# Run FastAPI app
python main.py
```
The backend API server will run at `http://localhost:8000`.

### 2. Run the Frontend

Navigate to the `frontend/` directory, install Node modules, and start the Vite development server:

```bash
cd frontend

# Install Node dependencies
npm install

# Start Vite dev server
npm run dev
```
The web application will open at `http://localhost:5173`. Any requests to `/api` will be proxied automatically to the backend at `http://localhost:8000`.

## Using Legacy CLI & Streamlit Tools

All original standalone CLI scripts have been moved to the `backend/tools/` directory.

To run these tools:
1. Make sure your backend virtual environment is active.
2. Run them from within the `backend/tools/` folder.

```bash
cd backend/tools

# Register faces via Webcam CLI
python add_faces.py

# Run local OpenCV camera window attendance recorder
python test.py

# Run local Streamlit dashboard
streamlit run app.py
```
*Note: Any CSV logs created by `test.py` or read by `app.py` are saved in `backend/tools/Attendance/`.*
